Red Hat Satellite

Red Hat Satellite is a tool that makes it easier for system
administrators to manage RHEL at scale.

Satellite provides system administrators with the tools they
need to automate tasks and be more efficient in the work
that they do.

This includes giving them tools to manage:

  • Patching & Lifecycle Management
  • Provisioning of New Machines
  • Configuration
  • Security Compliance
  • Red Hat Subscription Management
